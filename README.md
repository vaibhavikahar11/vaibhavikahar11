## Featured Projects

### 🚀 CryptoNex — AI-Powered Crypto Trading Platform
**Tech:** React, Spring Boot, Node.js, Express.js, Gemini AI, Razorpay, Stripe
- Designed a microservices architecture with React, Spring Boot, and Node.js, unified via a custom Express.js API Gateway for centralized routing and CORS handling
- Built an AI chatbot using the Google Gemini API with live CoinGecko data injected into prompts for real-time, context-aware crypto market responses
- Implemented JWT authentication, 2FA, and email OTP (Spring Security + Java Mail Sender), along with Razorpay and Stripe payment integrations for wallet top-up, transfers, and withdrawals

### 🔍 Vehicle Insurance Fraud Detection
**Tech:** Python, Scikit-learn, EDA, SMOTE
- Compared multiple classification models for fraud detection, prioritizing recall as the key metric; selected a Neural Network with Autoencoder achieving **95% recall**, a 33% gain over Random Forest
- Improved recall by 15–20% across models through SMOTE, normalization, and feature engineering, with a Voting Classifier achieving 92% recall and 18% F1-score
